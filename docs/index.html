What it does

Multi-broker support

Connect to Azure Service Bus and RabbitMQ from a single interface. More providers coming.

Browse and manage

View queues, topics, subscriptions and exchanges. Send, peek and resubmit messages with ease.

Dead-letter handling

Real-time notifications for dead-letter messages with click-to-navigate and resubmit support.

Pattern detection

Automatic detection of RabbitMQ messaging patterns: Work Queue, Pub/Sub, Routing, Topics, RPC, Headers.

Cross-platform

Native desktop apps for Windows, macOS and Linux. Plus a web mode for browser access.

Free and open source

MIT licensed. No accounts, no telemetry, no subscription. Use it, fork it, ship it.
